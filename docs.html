总览

对象:每条都是机械臂夹爪 / 人手做 manipulation 的短视频,配一条 instruction + init_frame(首帧 = 条件帧)。

两个独立轴,各自独立打分:

  • Physical Adherence(物理真实度)PA — 视频本身物理是否真实(只看视频,不看 instruction)。
  • Instruction Alignment(指令对齐)IA — 视频是否完成了 instruction(结合 instruction + init_frame)。

每轴 = 1 个 1–5 主分(综合) + 3 个子项(客观,0/1/2 三档:✗ 0 = major / 违反 · ⚠ 1 = minor / 部分 · ✓ 2 = holds / 通过) + 必填 note(每个 ✗/⚠ 子项写一句视觉证据)。

⚠ 中间档必须用起来 — "左右手用错但动作完成"、"轻微穿模"、"目标基本但没完全完成" 这种典型 minor 问题不要打成 ✗ 或 ✓,要选 ⚠。

Axis A — Physical Adherence (1–5)

只凭视频判断物理真实性,与 instruction 无关。

3 个子项(✗ / ⚠ / ✓)

机械手/人手完整性 · agent_consistency
✓ 结构全程正常稳定。
⚠ 轻微短暂变形/抖动但仍可辨认为夹爪/手。
✗ 明显融化、扭曲、消失、多/少肢。
背景与物体一致性 · scene_consistency
✓ 背景 + 物体全程稳定一致。
⚠ 轻微闪烁/漂移、个别物体短暂不稳。
✗ 场景崩坏、物体瞬移/凭空增减/严重形变。
交互真实性 · interaction_realism
✓ 接触、抓取、受力合理。
⚠ 轻微穿模 / 接触略不自然但动作成立。
✗ 明显穿模、抓空却移动物体、违反重力惯性。

1–5 主分(综合,参考子项但非机械相加)

5物理高度真实 — 几乎挑不出毛病。
4物理大致真实 — 仅轻微瑕疵。
3物理明显不一致 — 可看但有明显瑕疵。
2物理严重违反 — 重大违反但场景可识别。
1物理完全崩坏 — 几乎全篇物理失效。

Axis B — Instruction Alignment (1–5)

用 instruction + init_frame 判断是否完成任务(评任务语义与完成度,不评画面真实度)。

3 个子项(✗ / ⚠ / ✓)

动作与主体匹配 · agent_match
✓ 对的主体做对动作。
⚠ 动作完成 + 主体小偏差(例:要求右手却用左手、用了相邻的手)。
✗ 无关动作 / 完全错误的主体。
目标物正确 · object_correct
✓ 操作的就是 instruction 指定的目标物体(物体对、颜色对)。
⚠ 部分正确 / 有歧义 / 碰到相邻物。
✗ 完全错误的物体。
目标完成度 · goal_completed
✓ instruction 目标终态完整达成。
⚠ 部分完成 / 接近但没完全(如举起但没放进去)。
✗ 没完成 / 完全跑偏。

1–5 主分(综合)

5指令完全符合 — 主体、物体、目标都对、完整达成。
4指令基本符合 — 主要完成,仅轻微欠缺(略歪 / 没完全松爪)。
3指令部分偏离 — 大方向对但有 1 个子项 ⚠ 或 ✗。
2指令明显不符 — 错动作或错物体,多个子项 ✗。
1指令完全不符 — 错主体 / 不动 / 无关动作。

判分要点 / 标注流程

  • 先打子项、再综合主分:对照每个 ✓/⚠/✗ 客观判断,主分参考子项但非机械相加。
  • ⚠ 中间档必须用起来:数据显示 IA 的 agent_match / object_correct 几乎只有 ✓ 或 ✗,几乎没人用 ⚠ — 不对。"左右手用错但动作完成"、"轻微穿模"、"目标基本完成但没完全" 都属于 ⚠。
  • 两轴独立:可以"物理完美但没按指令做"(PA 高 / IA 低),也可以"完成了任务但画面不真实"(IA 高 / PA 低)。
  • PA 只看视频:判物理时忽略 instruction。
  • IA 以 init_frame 为参照判断 agent / 场景起点。
  • notes 必填:每个标 ⚠ 或 ✗ 的子项写一句具体观察(视觉证据)。系统会按你打的分自动预填模板,你只需补充冒号后细节。
  • 默认 5 分 + 全 ✓:完美视频可直接保存。发现问题再调子项 + 主分。
  • 视频/数据有问题(指令与首帧不符 / 视频无法播放 / 视频静止黑屏 / 其他):**🚨 报错** 按钮 → 管理员处理,该条不再发给其他人。
  • 不确定 / 想下一题:跳过(题会回到队列最前面给别人)。

FAQ

  • Q:指令要求右手、视频用左手、其它都对怎么打?
    A:agent_match = ⚠(1),object_correct + goal_completed = ✓;IA 主分 3–4(不是 1)。这是典型 minor 问题。
  • Q:视频有轻微穿模、目标却完成了?
    A:interaction_realism = ⚠;goal_completed = ✓。PA 主分 3–4,IA 主分 4–5。
  • Q:主体对、物体对、动作对但还没到终态(只是放到一半)?
    A:goal_completed = ⚠;主分 IA = 3(部分完成)。

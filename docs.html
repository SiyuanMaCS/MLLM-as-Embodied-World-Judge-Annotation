Embodied World-Model Video Evaluation

对象:每条都是机械臂夹爪 / 人手做 manipulation 的短视频,配一条 instruction + init_frame(首帧=条件帧)。

两个独立轴,各自独立打分:

  • Physical Adherence(物理真实度):视频本身物理是否真实 — 只看视频,不看 instruction。
  • Instruction Alignment(指令对齐):视频是否完成了 instruction — 看 instruction + init_frame。

每轴 = 1 个 1–5 主分 + 3 个 0/1 小分 + 必填 note(共 8 字段全标)。

Axis A — Physical Adherence (1–5)

定义:只凭视频判断是否符合物理常识,与 instruction 无关。

3 个 0/1 小分(0 = 违反,1 = 成立)

agent_consistency · 机械手/人手完整性
夹爪/手全程结构完整、刚性、时序一致 — 不融化、手指不黏连、不多/少指、不橡皮形变、不闪烁。
1=手指形状稳定;0=抓取时手融成一团 / 多出一根手指。
scene_consistency · 背景与物体一致性
背景和被操纵物体时序连贯 — 不瞬移、相关物体不凭空出现/消失、不无故变形变色、背景不闪。
1=方块保持同一身份/大小/颜色;0=方块中途变色 / 凭空多一个物体。
interaction_realism · 交互真实性
接触与受力合理 — 夹爪/手真实闭合并支撑物体、无穿模、遵守重力惯性与合理受力传递。
1=物体被夹住随手移动、接触可承重;0=物体无支撑漂浮 / 手穿过杯子 / 物体逆重力跳动。

1–5 主分

1完全不符:普遍违反,机械手/场景崩坏(融化、瞬移、严重穿模),多项小分严重 fail。
2较差:部分物理成立但有重大违反(如物体无支撑漂浮,或手明显穿过物体)。
3中等:大体合理但有明显不一致(短暂手指形变、短暂穿模、轻微闪烁),观者会注意到。
4良好:物理上可信,仅轻微瑕疵(单个小 artifact、瞬间抖动)。
5完美:无可见物理违反,机械手/场景/交互都真实。

Example · physical_adherence=2 — agent_consistency fail:抓取时夹爪融成一团;interaction_realism fail:杯子短暂穿过桌沿;scene_consistency 成立。重大违反 → 2。

Axis B — Instruction Alignment (1–5)

定义:用 instruction + init_frame 判断视频是否完成任务(评任务语义与完成度,不评画面真实度)。

3 个 0/1 小分(0 = 否,1 = 是)

agent_match · 机械手匹配
完成任务的手/夹爪是不是 init_frame 里那个(不是新出现/瞬移的手臂,不是别的 agent)。
1=首帧的夹爪在做任务;0=另一只手臂出现来做。
object_correct · 物体正确
被操纵的是不是 instruction 指定的目标物体(物体对、属性对 — 颜色/类型)。
1="抓红方块"抓的就是红方块;0=抓成了蓝方块/别的物体。
goal_completed · 目标达成
instruction 的目标终态是否真正达成(如真把物体放进容器,而不是只是举起来靠近)。
1=终态在画面里达成;0=只做了一半 / 接近但没完成。

1–5 主分

1完全不符:错的机械手 / agent 不动 / 做无关动作(agent_match 通常 0)。
2错动作或错物体:对的机械手动了,但操作了错的物体或做了错的动作(如要左转却右转)。
3部分完成:对的机械手在对的物体上操作,但目标只部分达成(趋向/开始但没到终态,如举起方块却没放进箱子)。
4接近完成:目标基本达成,仅轻微欠缺(放了但略歪 / 没完全松爪)。
5完整完成:正确机械手在正确物体上完整、准确达成目标。

Example · instruction_alignment=3 — agent_match 成立(首帧夹爪在做)、object_correct 成立(抓的是正确红方块),但 goal_completed fail:只略微抬起、从未放进箱子。部分完成 → 3。

标注须知

  • 两轴独立:物理真实与指令对齐分开打;可以"物理完美但没按指令做"(物理高/对齐低),也可以"完成了任务但画面不真实"(对齐高/物理低)。
  • Physical 只看视频:判物理时忽略 instruction,只问"看起来真不真实"。
  • Instruction 以 init_frame 为参照判断 agent / 场景起点。
  • 8 字段全标(2 主 1–5 + 6 小分 0/1)。
  • ⚠ 小分交互方向:**勾选 = 此项被违背**(红 ✗);默认不勾 = 通过。底层存值仍 1 = 通过 / 0 = 违背(unchanged),只是输入方向倒过来让"标违背"更轻松。
  • note 必填(非 optional):简述打分的视觉证据,尤其是任何标 0 的小分。
  • 小分是证据、主分是综合:主分需与小分一致(如 goal_completed=0 时 Instruction Alignment 一般 ≤3)。

依据:VideoPhy / VideoPhy2、VideoScore / VideoScore2、WorldModelBench、WorldArena、VBench-2.0。
